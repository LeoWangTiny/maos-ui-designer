# Generic Design Skill Design Brief

## Audience

Operations users who need to monitor edge command activity and review important alerts.

## Goal

Create a polished web app page with clear sections, reusable design system components, and a professional dashboard layout.

## Design Direction

Use a design-system-oriented layout with a sidebar, summary metrics, a table, a chart, and an alert detail panel. Keep the surface responsive for desktop and mobile. Use a restrained blue and gray palette with warning and danger colors for priority items.

## Strengths

- Reusable page sections.
- Clear navigation and cards.
- Better hierarchy than a generic dashboard.
- Responsive layout and screenshot-friendly structure.

## Limits

This generic design-skill output does not fully define an AI-friendly implementation gate, TypeScript typed API, Ant Design ConfigProvider theme tokens, domain wrappers, command approval policy, receipt pending states, destructive confirmation, or two-person approval.
