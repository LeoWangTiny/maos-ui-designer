# MAOS UI Designer Prompt

Use `$maos-ui-designer` to design and implement a static prototype for an edge command platform Web B2B console.

The prototype must cover three real screens in one artifact:

- Command Center Overview
- Alert Triage
- Command Approval Drawer

It must apply industrial IoT context, command-center density, AI-friendly framework selection, Ant Design-style B2B components, semantic tokens, operational states, permission and safety boundaries, stable selectors, and responsive QA hooks.
