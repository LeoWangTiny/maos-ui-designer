# Control No Skill Design Brief

## Audience

General operations users who need a dashboard for edge command activity.

## Goal

Create a clean dashboard that shows platform status, node counts, alerts, recent tasks, and command activity.

## Design Direction

Use a modern dashboard style with a blue accent, summary cards, charts, and a table. Keep the page simple and visually attractive.

## Notes

This control sample intentionally avoids MAOS-specific guidance. It does not define an industry context matrix, AI-friendly framework gate, semantic token strategy, domain wrappers, operational state model, or explicit B2B component architecture.
